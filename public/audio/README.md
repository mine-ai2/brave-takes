# Meditation Audio Files

This folder should contain royalty-free meditation/ambient audio loops.

## Required Files

Add the following MP3 files:
- `calm-waves.mp3` - Ocean/wave sounds
- `forest-morning.mp3` - Forest ambience with birds
- `gentle-rain.mp3` - Rain sounds
- `soft-piano.mp3` - Gentle piano music
- `breathing-space.mp3` - Calm ambient sounds
- `zen-garden.mp3` - Asian-inspired ambient music

## Recommended Sources

1. **Pixabay Music** (https://pixabay.com/music/) - Free, no attribution required
2. **Free Music Archive** (https://freemusicarchive.org/) - CC licensed
3. **Mixkit** (https://mixkit.co/free-stock-music/) - Free for commercial use
4. **YouTube Audio Library** (studio.youtube.com) - Free for creators

## Requirements

- MP3 format
- Should loop seamlessly
- 1-5 minutes length is fine (they loop)
- Calming, not distracting
- No vocals

The app will gracefully handle missing audio files - the Reset step will still work without audio.
